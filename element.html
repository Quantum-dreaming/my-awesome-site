常见HTML标签

链接和图片
<a href="https://example.com">点这里</a>
<img src="cat.jpg" alt="喵图">
点击进入bilibili
<p>：段落标签

基础块，显示一个段落。

<p>This is a paragraph.</p>

This is a paragraph.

<h1>~<h6>：标题标签

从最大到最小一共6级。

<h1>大佬级标题</h1>
<h6>边缘人的标题</h6>

大佬级标题

迷你标题
<a>：超链接

点击跳转至另一个页面，不加 href 网页会很单调

<a href="https://example.com">点我去下一个板块/你想浏览的界面。</a>
<img>：图片

丰富页面内容，可以加 src 和 alt组合为

<img src="cat.jpg" alt="一只可爱小猫">

链接和图片开始已有示例，不再重复


<ul> / <ol> / <li>：列表三件套

<ul> 是无序<ol> 是有序<li> 是列表项

<ul>
  <li>咕咕咕</li>
  <li>继续咕</li>
</ul>
  • 咕咕咕
  • 继续咕
<strong> / <em>：强调标签

<strong> 程度较大,效果为加粗<em> 程度较小，效果为斜体

<p>这很 <strong>重要</strong>！</p>
<p>我 <em>真的</em>生气了。</p>

这很 重要！

我 真的 生气了。

<div> / <span>: 这俩是万能块

<div> 是块级<span> 用于处理行内内容

<div>一个区域</div>
<p>这是 <span>加了样式</span> 的文字</p>

如本页中每一个代码块均为div块

<br> / <hr>

<br> 是换行，<hr> 是分割线

一行文字<br>下一行文字<hr>

此页中每一个代码块示例后都在悄悄换行哦(不推荐学习！！！在css文件中为容器设置内边距更佳)

我是分割线

<input> / <textarea> / <button>：三种表单

收集用户信息的工具包（天天用来让你输验证码的那种）

<input type="text" placeholder="请输入你的姓名"><br>
<textarea>你想说点什么呢？</textarea><br>
<button>提交</button>
